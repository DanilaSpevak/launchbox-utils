# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a Vulnerability

If you discover a security issue, please report it privately:

1. Open a [GitHub Security Advisory](https://github.com/DanilaSpevak/launchbox-utils/security/advisories/new) for this repository, or
2. Contact the maintainer through GitHub.

Please do not open a public issue for undisclosed security vulnerabilities.

Include:

- A description of the issue and its potential impact
- Steps to reproduce
- Affected commands or GUI flows
- Your environment (Windows version, Python version)

We will acknowledge reports as soon as possible and coordinate a fix before public disclosure when appropriate.
